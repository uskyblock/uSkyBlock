# Configuration Reference

This page shows the **default `config.yml`** shipped with uSkyBlock, split into
sections for easy navigation.  Every key is documented with inline comments
explaining its type, accepted values, and default.

Copy sections directly into your `plugins/uSkyBlock/config.yml` — only include
the keys you want to change. You can also view the
[full raw config.yml on GitHub](https://github.com/uskyblock/uSkyBlock/blob/master/uSkyBlock-Core/src/main/resources/config.yml).

!!! tip
    Use the **sidebar** to jump to a section, or `Ctrl+F` / `Cmd+F` to
    search for a specific key.
